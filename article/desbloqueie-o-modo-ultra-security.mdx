Já imaginou um desconhecido tentar acessar o seu computador e toda vez que digitar a senha corretamente ou até mesmo
entrar automaticamente, ele apenas desligar? Seria estranho, certamente, você poderia dizer até que é um vírus,
mas eu vou explicar para você.

## Modo Ultra Security

Imagine que você que vai sair de férias e você não gosta de deixar seu computador com senha e tem receio
de sua família ou algum primo distante chegar e usar o seu computador, detonar ele ou até mesmo acessar
documentos que não deveriam. A solução se chama Ultra Security.

Como funciona? É simples. Eu desenvolvi um script que irá rodar junto da pasta "Inicializar" ou "Startup",
e ele irá aguardar dez segundos para encontrar uma pasta chamada `iamhere`, caso ela não exista,
o computador irá executar o desligamento automaticamente. Interessante, né?

## Como faço para instalar?

Primeiramente, vá até o meu Github e faça o [download do script](https://github.com/Yagasaki7K/feature-ultrasecurity).
É só clicar no botão verde `< > Code`, e fazer o download do zip.

Com o zip no seu computador. Descompacte ele. E você verá dois arquivos. Um é o `Ultra Security.exe`,
e o outro é um atalho para o "Startup", a página de inicialização do Windows.

Agora, o que você precisa fazer, é literalmente, arrastar o programa `Ultra Security` para dentro do `Startup`
e toda vez que o seu computador iniciar, o nosso programa irá executar junto.

Caso já queira ver o resultado, é só reiniciar o computador e voilà!

Vale lembrar que essa ferramenta não é indicada para uso diário, mas é extremamente útil quando você vai se ausentar por
alguns dias e não quer que ninguém mexa no seu computador.

Caso queira entender o que o `Ultra Security.exe` está fazendo, vou deixar o código abaixo:

```bash
; For the tutorial and more details, visit: yagasaki.dev

#NoEnv  ; Recommended for performance and compatibility with future AutoHotkey releases.
; #Warn  ; Enable warnings to assist with detecting common errors.
SendMode Input  ; Recommended for new scripts due to its superior speed and reliability.
SetWorkingDir %A_ScriptDir%  ; Ensures a consistent starting directory.

; Wait for 10 seconds
Sleep, 10000

; Get the path to the desktop
desktopPath := A_Desktop "\iamhere"

; Check if the "iamhere" folder exists on the desktop; if not, shutdown the computer
If !FileExist(desktopPath)
{
    Shutdown, 5
}
```
